\section{研究目标与创新点}

\begin{frame}{研究目标与创新点}
  \begin{columns}[T,onlytextwidth]
    \column{0.48\textwidth}
    \begin{block}{研究目标}
      \begin{itemize}
        \item 搭建\hlb{可复现实验平台}，统一比较检索、重排、补检索与验证模块。
        \item 定量评估\hl{异构接入}、\hl{Adaptive PRF}、\hl{CoVe} 三类机制的净贡献。
        \item 诊断它们在\hl{效果}、\hl{成本}、\hl{安全}之间的耦合关系。
      \end{itemize}
    \end{block}

    \column{0.52\textwidth}
    \begin{block}{三点创新}
      \begin{enumerate}
        \item \hlb{统一证据单元}：以 \texttt{EvidenceUnit} 统一文本、表格、图谱的检索入口。
        \item \hlb{自适应补检索}：以重排序分数作为置信信号，低置信时触发查询扩展与二次检索。
        \item \hlb{后验验证闭环}：生成后做声明级核验，证据不足时输出 \hl{No-Answer}。
      \end{enumerate}
    \end{block}
  \end{columns}

  \vspace{0.4em}
  \begin{alertblock}{本文的贡献重点}
    贡献不在于证明系统已经最优，而在于\hl{揭示模块叠加后的副作用与权衡边界}。
  \end{alertblock}
\end{frame}
